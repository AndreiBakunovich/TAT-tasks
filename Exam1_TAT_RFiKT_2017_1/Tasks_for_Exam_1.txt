Задача 1
Написать программу для валидации пути некоторого объекта файловой системы в OC Windows. Имя вводится в качестве аргумента командной строки.
Не пользоваться регулярными выражениями.

Задача 2
Для содержащихся в некоторой коллекции данных (объектов) подсчитать и вывести количество повторящихся.

Задача 3
Написать программу, позволяющую осуществлять последовательный ввод с консоли произвольного числа автомобилей, включающих следующие поля:
1. Марка (string) (mark)
2. Модель (string) (model)
3. Тип (седан, универсал, внедорожник) 
sedan
multi-purpose vehicle (MPV) 
sports utility vehicle (SUV)
4. Цена (price)
После окончания ввода выполнить вывод списка автомобилей, отсортированный сначала по цене, далее в рамках выполненной сортировки отсортировать по типу и далее марке.

Задача 4
Написать игру “Морской бой”. 
На поле 10X10, каждая клетка которого определяется букво-цифровой координатой (число по вертикали и буква по горизонтали), находится случайное количество кораблей, расставленных случайным образом. Каждый корабль занимает одну клетку и не может быть расположен в соседней клетке с другим кораблем. Если введенная с консоли координата совпадает с координатой корабля, он считается потопленым. Игра продолжается до тех пор, пока все корабли не будут потоплены. 
По окончании игры выводится количество потопленных кораблей и количество потребовавшихся выстрелов.
